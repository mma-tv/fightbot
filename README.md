Fight Bot
=========

MMA/Boxing fight logger and polling script for Eggdrop bots


Requirements
------------

  - Eggdrop 1.6.16+
  - TCL 8.5+
  - SQLite 3.6.19+


Setup
-----

  1. Install SQLite
     * *(instructions later)*
  2. Copy fights.tcl and util.tcl to eggdrop scripts directory
  3. Copy fights.sql to eggdrop directory
